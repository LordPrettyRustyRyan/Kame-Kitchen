# Kame Kitchen  
Innovative Recipe & Food Blog Platform
Kame Kitchen is a full-stack recipe website focused on innovative cooking, clean visual design, and structured food content.  
The platform currently fetches recipe data dynamically from the backend with clear steps, ingredients, accessories, and nutritional values through a visually rich and responsive UI.

---

## Tech Stack
### Frontend
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Font Awesome
- jQuery
### Backend
- PHP
- MySQL (phpMyAdmin)

## Current Features
- Dynamic recipe rendering from database (PHP + MySQL)
- Detailed recipe pages with:
  - Ingredients list
  - Accessories list
  - Step-by-step cooking instructions
  - Nutritional values
- Clean, modern, and visually rich UI
- Fully responsive design (desktop, tablet & mobile)
- Backend-powered recipe data fetching
- Optimized layout and typography for content readability

## Planned Enhancements - Future Scope
- Backend-powered blog system (PHP + MySQL)
- Convert all pages into database-backed PHP pages
- Pagination for recipes and blog listings
- Category-based filtering
- Admin panel for managing recipes and blogs
- Performance optimizations and query refinement

> The project is intentionally structured for scalability and future feature expansion.
